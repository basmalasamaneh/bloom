# Bloom – Web Platform

Bloom is a smart agriculture platform designed to help farmers manage crops, access expert advice, and interact with an integrated agricultural marketplace also interact with each other in the community — all in one system.

The web platform provides a complete dashboard experience for farmers, experts, and administrators.

---

## Features

### For Farmers & Users
- Crop case management (add crops, track growth stages)
- AI-based recommendations and daily farming tasks
- Plant disease detection using image analysis
- Real-time chat with farmers and experts
- Community interaction (posts, comments, sharing)
- Marketplace (buy, sell, rent agricultural tools/products)
- Order & rental tracking system

### AI Capabilities
- Smart crop life-cycle generation
- Environmental condition validation
- AI farming tips based on weather & crops
- Image-based plant health analysis

### Marketplace
- Buy / Sell / Rent products
- Product Q&A system
- Order tracking (Pending → Delivered → Archived)
- Reporting system for safety

### Communication
- Real-time messaging (WebSockets)
- Paid expert consultation system

### Admin Dashboard
- User management & moderation
- Expert verification system
- Product & marketplace control
- Reports handling (users & posts)

## Project Goal

Bloom aims to simplify farming workflows by integrating multiple agricultural services into one platform instead of using separate systems.
